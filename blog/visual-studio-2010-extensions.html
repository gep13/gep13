Visual Studio 2010 Extensions

Published on 15 January 2011
extensions microsoft visual studio 2010

Having completed a new install of Visual Studio 2010 I thought I would keep a note of the Extensions that I have added to the default installation, that way I won’t have to try and remember them all if I have to do another install. My memory is bad enough as it is without having to remember anything else!!

  • All-In-One Code Framework Sample Browser (more info can be found here)
  • Amazon Web Services Toolkit for Visual Studio
  • CKS - Development Tools Edition (Foundation)
  • Color Printing
  • CSSCop
  • ExtensionSync (more info can be found here)
  • GhostDoc
  • GoOffline
  • Hide Main Menu
  • JScript Editor
  • KillCassini
  • NuGet Package Manager
  • PowerCommands for Visual Studio 2010
  • Presentation Zoom
  • Productivity Power Tools
  • Remove Unused References
  • Silverlight SharePoint Web Parts
  • SlowCheetah - XML Transforms
  • Snippet Designer
  • StyleCop
  • Team Foundation Server Power Tools
  • Visual Studio 2010 SharePoint Power Tools
  • Visual Studio Color Theme Editor
  • Windows Phone Test Project
  • World of VS Default Browser Switcher
  • World of VS Quick Add Reference

Are there any others that I should get?

Note: I will be updating this blog post as and when I add any other extensions.
